# Calculadora-matematica-em-c
Projeto simples de uma calculadora desenvolvida em C++.

# Sobre o projeto
Este é um projeto criado com o objetivo de praticar conceitos básicos de programação, como:

  - Operações matemáticas
  - Estruturas condicionais
  - Funções
  - Entrada e saída de dados no terminal
  - Organização de código em C++

A aplicação funciona diretamente no terminal e realiza operações matemáticas básicas de forma simples e prática.

# Tecnologias utilizadas
  - C++

# Como executar
Compilar o arquivo .cpp utilizando um compilador C++.
